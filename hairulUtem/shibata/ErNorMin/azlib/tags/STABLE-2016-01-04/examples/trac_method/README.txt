力法による剛性最大化と体積最小化のサンプルプログラム
====================================================

このサンプルプログラムは，力法による剛性最大化，または体積最小化を行う
サンプルプログラムである．

exampleにあるサンプルプログラムとしてはかなり複雑な部類に入る．
パラメータファイル，BDF(Nastran), DXF, STl, MGF(AVS), INP(AVS)などの
I/Oに関する関数，FEMソルバー，そして最適化の統合プログラムになっている．

元来，AzLibはこのような形状最適化プログラムを作るための基礎を共通化する
ことを主目的に作られたライブラリである．つまり，このプログラムが形状最適
化に関する研究をする場合の基盤(もしくはテンプレート)となる．


使い方
======
Linuxであれば，run.shを実行するだけでコンパイルから実行まで自動的に
行われる.

Windowsでは
 nmake -f Makefile.win
 trac_method.exe PARAM_bodyless bar.bdf bar_rest.bdf bar_bodyless
のように実行すれば，最適化のプログラムが実行される．(未確認, 2006-01-23)

30回まで形状更新をして停止するが，CPUの性能によって数分はかかる．


引数の説明
==========
 [実行ファイル] [パラメータファイル] [オリジナル形状] [形状拘束] 
 [結果ファイル名]

パラメータファイル :
 各種パラメータを記述したファイル．サンプルとして，PARAM_bodyless(体積最小化),
 PARAM_Rmax(剛性最大化)が用意されている．

オリジナル形状 :
 Nastranのバルクデータファイル(*.bdf, *.NAS)
 荷重，拘束などの境界条件が必要．

形状拘束 :
 オリジナル形状のファイルに形状変更しない部分の拘束条件を追加したファイル

結果ファイル名 :
 形状最適化後の結果を出力する時のファイル名．
 例えば，「aaa」とすると，aaa_trac_result.bdf というファイルができる．
 上記の実行例では bar_bodyless を指定している．
