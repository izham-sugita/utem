# AzLib

 $Id: README.txt 1037 2016-02-05 11:11:47Z hayashi $

C言語による最適化プログラムの開発環境．

-------------------------------------------------------------------------------

* Makefile
- UNIX 向け Makefile

* doc/
- AzLib 開発に当っての必要になる資料，メモ等 (未整理)

* examples/
- AzLib の使いかたのサンプル (未整理)

* projects/
- Microsoft VisualStudio 用のプロジェクトファイル．
  eclipse, Boland等のプロジェクトファイルを入れても良い．

* src/
- AzLib本体

* test/
- AzLib内部の関数を実装する時のテストプログラム等
  プロトタイプやベンチマークプログラム，テストケースも含む．

-------------------------------------------------------------------------------

* AzLibを使ったプログラム開発用に，各研究内容ごとのライブラリを作る．
* 最適化実行中もモデルを表示しておき，形状更新をリアルタイムで見れないか．
