# 有限要素モデルビューアー

 $Id: README.txt 1067 2016-05-25 09:51:24Z hayashi $


はじめに
--------
このサンプルプログラムは，NASTRANのバルクデータフォーマットで記述されている有限要
素モデルのファイルを読み込み，OpenGLを用いてディスプレイ上にモデルを表示するもの
である．つまり，簡易的な有限要素モデルビューアーである．
単純にバルクデータファイルを表示させたいことがときどきあるため，サンプルプログラ
ムとして用意した．


使い方
------
- 以下の引数の説明を参考にして，直接実行する．あるいは，run.shの該当部分を書き換
  えて実行する．
- 終了するには，'q' を入力する．


引数の説明
----------

./main [形状データファイル]

- 形状データファイル：
	Nastranのバルクデータファイル（*.bdf，*.NASなど）
